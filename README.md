# CV

## Commits title format

Commit title are build like following :
`<Tag> <Description>`

Example: `[ADD] Ability to customize color through url param`

| Tag       | Definition             |
|-----------|------------------------|
| [ADD]     | New feature added      |
| [UPD]     | Feature updated        |
| [FIX]     | Bug fixed              |
| [MV]      | Code moved             |
| [DEL]     | Code/feature removed   |
| [STYLE]   | Display modified       |
| [CONTENT] | Content modified       |
| [DOC]     | Documentation modified |
